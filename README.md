<p>This project is licensed under the GNU General Public License v3.0.</p>
<p>This code is going to be used in the Cuck and Cock visual novel game</p>
<h2>Installation</h2>
<p>cmake .</p>
<p>make</p>
<p>to run - ./lcnovel or CCNVL_FILE=file.ccnvl FONT=optionalfont.ttf ./lcnovel</p>
<p>to create a new ccnvl file see web_generator/ </p>


<h2>To create ccnvl project:</h2>
<p>create folder of the project with all data included</p>
<p>by using web page generator.html create scripts in .txt format with novel commands (see more in web_generator/README.md)</p>
<p>use web_generator/ccnvl_constructor.py with correct params to create new ccnvl file runnable by lcnovel</p>
